# Nitin Shivakumar — Portfolio

A single-page portfolio site showcasing experience, projects, and skills in Data Science, ML, and Gen AI.

## Contents

- **Hero** — Name, tagline, and links (GitHub, Data Science Portfolio, LinkedIn, Resume)
- **Projects** — Filterable GitHub projects (Gen AI/RAG, ML & Data, Computer Vision)
- **Experience** — Expandable career timeline (Genpact, Aptean)
- **Skills** — Animated skill bars (Programming, Platforms, Methods)
- **Education** — University at Buffalo (MS Data Science), GITAM (B.E.)
- **Contact** — Email, phone, GitHub, Data Science Portfolio, LinkedIn

## How to view

1. Open `index.html` in a browser (double-click or drag into a browser window).
2. Or from the project folder run:
   ```bash
   open index.html
   ```
   (macOS) or use any local static server (e.g. `npx serve`, `python -m http.server`).

## Tech

- Plain **HTML**, **CSS**, and **JavaScript** — no build step or frameworks.
- Fonts: [Outfit](https://fonts.google.com/specimen/Outfit), [JetBrains Mono](https://www.jetbrains.com/lp/mono/) (Google Fonts).
- Resume: `Nitin_Shivakumar_DS_Resume.pdf` in this folder.
